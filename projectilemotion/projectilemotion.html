拋體運動操作說明｜How to Explore

本實驗固定以 y = 0 表示地表。設定參數後按下「解算」，軌跡才會以動畫方式從 P1 繪製至地表落點 T。

  1. 設定初速 V₀ 初速越大，飛行時間與水平位移X通常越大。
  2. 調整發射角 α α 是相對水平面的角度，會改變水平與垂直初速度的比例。
  3. 改變發射點高度H（Y軸） H 是發射點在 Y 軸上相對 y = 0 地表的高度；提高 H 會增加落地時間。
  4. 判讀結果 黃色 P1 點是發射點，綠色 T 點是 y = 0 的地表落點；X 表示水平位移X，Y 表示垂直位移（海拔）。
  5. 執行解算 參數變更會清除舊軌跡；按下「解算」後才播放青色拋物線動畫，完成時顯示 T 點與結果。
  6. 縮放與平移畫面 使用滑鼠滾輪或畫面上的 −、＋ 按鈕無段縮放；按 1:1 恢復預設比例，長按滑鼠左鍵拖曳可平移畫面，按 ⤢ 可重新適合目前軌跡。
  7. 切換參數控制區 桌面版可按左右區域分界上的箭頭，將右側參數控制區向右收合；按收合按鈕上方的 ‹ 可向左全螢幕顯示，放大文字並透過半透明底色保留左側解算展示區域的視覺參照；全螢幕時按 › 可退出。

Kinematics variants

三個基本公式與本頁變體

通式中的 v、v₀、a 與 x 都先指定同一個方向；本頁取向右與向上為正，水平方向 ax = 0，垂直方向 ay = −g，垂直初始位置為 H。

TRIG 初速三角分解

Vx = V₀‧cos α
Vy = V₀‧sin α

先把 V₀ 分解到水平與垂直方向，得到結果卡 01、02，並供後續四張卡使用。

BASE 01 v = v₀ + a‧t

vy(t) = Vy − g‧t
0 = Vy − g‧t最高

最高點的垂直速度為 0，因此得到結果卡 04：t最高 = Vy/g。

BASE 02 x = v₀‧t + 12a‧t²

Y = H + Vy‧t − 12g‧t²
X = Vx‧t

令落地時 Y = 0 並取非負時間根，得到結果卡 03；水平方向因 ax = 0，直接得到結果卡 05。

BASE 03 v² = v₀² + 2‧a‧x

0 = Vy² − 2‧g‧(Y最高 − H)

最高點令 vy = 0，解得結果卡 06：Y最高 = H + Vy²/(2‧g)。

快速示範：按下「示範組合」可切換到一組高處發射的明顯彈道。

地表拋體運動｜Projectile Motion to Ground

本頁採用理想化模型：忽略空氣阻力，重力加速度固定為 9.8 m/s²，並固定以 y = 0 表示水平地表。

右側參數控制區的「解算結果」會隨滑桿同步更新，方便直接比較參數與結果。

  • 座標約定：x 軸沿水平方向，y = 0 是地表；正 y 值可等效理解為海拔或相對地表的垂直距離。
  • 速度分量：水平初速度為 V₀‧cos α，垂直初速度為 V₀‧sin α，角度 α 一律以度表示。
  • 落地條件：令 y(t) = H + V₀‧sin α‧t − 4.9‧t² = 0，取非負時間根作為落地時間。
  • 位移觀念：X 表示從發射點到落點的水平位移X；H 與軌跡上的 Y 值表示垂直位移（海拔），最高值以最高點垂直位移Y（海拔）表示。
  • 解算流程：參數只即時更新公式；按下「解算」才繪製軌跡與 T 點。座標視野由使用者縮放或拖曳控制，不會因調整參數而自動改變。
