拋體運動操作說明｜How to Explore

本實驗固定以 y = 0 表示地表。P、Q 各自設定參數並按下對應的軌跡解算按鈕後，兩條軌跡才會以動畫方式繪製，且能保留在同一座標中重疊比較。

  1. 設定 P、Q 初速度 P：VP0 與 Q：VQ0 並列控制；初速越大，飛行時間與水平位移X通常越大。
  2. 調整發射角 αP、αQ 發射角是相對水平面的角度，會改變各組水平與垂直初速度的比例。
  3. 改變發射點高度H（Y軸） HP、HQ 是兩個發射點相對 y = 0 地表的高度；提高發射點通常會增加落地時間。
  4. 判讀結果 黃色 P 點與紫色 Q 點是發射點；TP、TQ 是兩組軌跡位於 y = 0 的地表落點。X 表示水平位移X，Y 表示垂直位移（海拔）。
  5. 分別執行解算 按「P軌跡解算」播放青色軌跡，按「Q軌跡解算」播放大紅色軌跡。修改其中一組參數只會清除該組舊軌跡，另一組結果會保留。
  6. 縮放與平移畫面 使用滑鼠滾輪或畫面上的 −、＋ 按鈕無段縮放；按 1:1 恢復預設比例，長按滑鼠左鍵拖曳可平移畫面，按 ⤢ 可重新適合目前軌跡。
  7. 切換 P、Q 點線段 左側解算展示區域下方共有四個開關，P、Q 各可整批切換所有向量與拋物線，或單獨切換 VP0、VQ0 的 X、Y 分量；這些操作只改變線段顯示，不會清除參數、公式數值或解算結果。
  8. 切換參數控制區 桌面版可按左右區域分界上的箭頭，將右側參數控制區向右收合；按收合按鈕上方的 ‹ 可向左全螢幕顯示，放大文字並透過半透明底色保留左側解算展示區域的視覺參照；全螢幕時按 › 可退出。

Kinematics variants

三個基本公式與本頁變體

通式中的 v、v₀、a 與 x 都先指定同一個方向；本頁取向右與向上為正，水平方向 ax = 0，垂直方向 ay = −g。下列 A 代表 P 或 Q，垂直初始位置分別為 HP、HQ。

TRIG 初速三角分解

VA,x = VA0‧cos αA
VA,y = VA0‧sin αA

分別把 P、Q 初速度分解到水平與垂直方向，得到結果卡 01、02 的並列結果，並供後續四張卡使用。

BASE 01 v = v₀ + a‧t

vA,y(t) = VA,y − g‧t
0 = VA,y − g‧tA,最高

最高點的垂直速度為 0，因此得到結果卡 04：tA,最高 = VA,y/g。

BASE 02 x = v₀‧t + 12a‧t²

YA = HA + VA,y‧t − 12g‧t²
XA = VA,x‧tA

令落地時 Y = 0 並取非負時間根，得到結果卡 03；水平方向因 ax = 0，直接得到結果卡 05。

BASE 03 v² = v₀² + 2‧a‧x

0 = VA,y² − 2‧g‧(YA,最高 − HA)

最高點令 vA,y = 0，解得結果卡 06：YA,最高 = HA + VA,y²/(2‧g)。

快速示範：按下「示範組合」可切換到一組適合比較 P、Q 軌跡的參數。

地表拋體運動｜Projectile Motion to Ground

本頁採用理想化模型：忽略空氣阻力，重力加速度固定為 9.8 m/s²，並固定以 y = 0 表示水平地表。

右側參數控制區的「解算結果」會隨 P、Q 滑桿同步更新，並在每張卡內並列顯示兩組公式與數值。

  • 座標約定：x 軸沿水平方向，y = 0 是地表；正 y 值可等效理解為海拔或相對地表的垂直位移。
  • 速度分量：A 代表 P 或 Q；VA,x = VA0‧cos αA，VA,y = VA0‧sin αA，角度一律以度表示。
  • 落地條件：分別令 yA(t) = HA + VA0‧sin αA‧t − 4.9‧t² = 0，取非負時間根作為各自落地時間。
  • 位移觀念：XP、XQ 表示各發射點到落點的水平位移X；HP、HQ 與軌跡上的 Y 值表示垂直位移（海拔）。
  • 解算流程：參數即時更新公式；按下 P 或 Q 軌跡解算才繪製該組軌跡與落點。兩組已解算資料可同時保留，座標視野也不會因調整參數而自動改變。
