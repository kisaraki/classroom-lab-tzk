## 一、適用規則

1. 循環系統的繁體中文專有名詞以本文件與上述教材為準。
2. 玩家可見文字、替代文字、Location、現行規格與測試必須使用教材詞彙。
3. 英文程式 ID 為既有資料契約，可在不顯示給玩家的前提下保留；不得把 ID 直譯成教材未使用的中文。
4. 關卡名稱的括號內容是遊戲路線或難度說明，不得被描述為教材新增的循環類型。
5. 遊戲顏色只是視覺編碼；血液狀態一律稱為「充氧血」或「減氧血」，不得以顏色取代生物學名稱。

## 三、標準詞彙

### 3.1 循環與血液狀態

| 標準詞 | 專案使用規則 |
| --- | --- |
| 血液循環 | 指血液在心臟與血管中的循環路徑 |
| 體循環 | 第一關與第三關的循環類型 |
| 肺循環 | 第二關與第四關的循環類型 |
| 冠狀循環 | 只在說明心臟本身的血液供應時使用 |
| 充氧血 | 肺泡微血管完成氣體交換後，以及左心與體循環交換前的血液狀態 |
| 減氧血 | 體微血管完成物質與氣體交換後，以及右心與肺循環交換前的血液狀態 |

不得以「含氧血」、「氧合血」、「缺氧血」、「去氧血」、「脫氧血」或「氧合狀態」取代「充氧血／減氧血」。病理敘述中的「缺血、缺氧」不屬於血液狀態名稱，不在此替換規則內。

### 3.2 基本路徑

教材的共同路徑順序為：

`心室 → 動脈 → 微血管 → 靜脈 → 心房`

| 循環 | 本案標準路徑 |
| --- | --- |
| 體循環 | 左心室 → 主動脈及其分支 → 體微血管 → 靜脈 → 上、下大靜脈 → 右心房 |
| 肺循環 | 右心室 → 肺動脈 → 肺泡微血管 → 肺靜脈 → 左心房 |

### 3.3 血管與交換區

| 標準詞 | 不採用或受限詞 | 說明 |
| --- | --- | --- |
| 微血管 | 毛細血管 | 教材在本章統一使用「微血管」 |
| 體微血管 | 組織微血管 | 體循環中進行物質與氣體交換的微血管 |
| 腹部及下肢的微血管網 | 下半身微血管、組織微血管 | 第一關 Location |
| 頭部、胸部及上肢的微血管網 | 腦循環、腦／上半身微血管 | 第三關 Location；仍屬體循環 |
| 肺泡微血管 | 肺交換區 | 第二、四關氣體交換 Location |
| 小動脈 | 細動脈 | 本教材用詞 |
| 小靜脈 | 細靜脈 | 本教材用詞 |
| 主動脈、主動脈分支 | 大動脈作為主動脈專名 | 關卡由左心室進入主動脈及其分支 |
| 上大靜脈、下大靜脈 | 上腔靜脈、下腔靜脈 | 本教材用詞 |

### 3.4 心臟

| 標準詞 | 使用方式 |
| --- | --- |
| 右心房、右心室、左心房、左心室 | 四個心臟腔室的名稱 |
| 心室中膈 | 分隔左、右心室的構造 |
| 瓣膜 | 防止血液逆流的構造總稱 |
| 房室瓣 | 心房與心室之間的瓣膜 |
| 二尖瓣 | 左心房與左心室之間的房室瓣 |
| 三尖瓣 | 右心房與右心室之間的房室瓣 |
| 半月瓣 | 心室與相連動脈之間的瓣膜 |

## 四、遊戲路線命名

| Level | 玩家可見名稱 | 交換 Location |
| ---: | --- | --- |
| 1 | 體循環（腹部及下肢） | 腹部及下肢的微血管網 |
| 2 | 肺循環 | 肺泡微血管 |
| 3 | 體循環（頭部、胸部及上肢） | 頭部、胸部及上肢的微血管網 |
| 4 | 肺循環（高危險關卡） | 肺泡微血管 |

第四關不是第三種循環；「高危險關卡」只表示遊戲難度。四關完成時應稱為「四關循環任務完成」，不得稱為「四循環完成」。

## 六、維護檢查

修改循環內容時至少檢查：

1. Location 與關卡名稱是否符合第四節。
2. 是否只使用「充氧血／減氧血」描述血液狀態。
3. 是否只使用「微血管」而非「毛細血管」。
4. 是否把第一、三關維持為體循環，把第二、四關維持為肺循環。
5. 氣體交換是否只位於體微血管或肺泡微血管。
6. 內部英文 ID 是否仍正確映射到玩家可見的教材詞彙。
